# 🍽️ Ember & Olive – Premium Restaurant Landing Page

A modern, elegant, and fully responsive restaurant landing page built with **HTML**, **CSS**, and **Vanilla JavaScript**. Designed for a luxury wood-fired Mediterranean restaurant, this project focuses on creating an immersive user experience through premium typography, handcrafted SVG illustrations, smooth animations, and a clean, modern layout.

## ✨ Features

- Responsive design for all screen sizes
- Premium hero section with custom SVG fire illustration
- Sticky navigation bar
- Interactive menu tabs
- Philosophy & storytelling section
- Custom SVG gallery illustrations
- Reservation form UI
- Smooth scrolling and animations
- Elegant dark theme
- Clean and semantic HTML5 structure
- Lightweight and fast (No frameworks)

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- SVG
- Google Fonts (Fraunces & Manrope)
